# Frontend - Store Ratings Platform

This is the **frontend** of the Store Ratings Platform project.  
The frontend is built using **ReactJS** and provides a professional, role-based interface for users, store owners, and admins.

---

## **Tech Stack**

- **Frontend:** ReactJS  
- **Routing:** React Router  
- **Styling:** Inline CSS with professional gradients and card layouts  
- **State Management:** React Hooks (useState)

---

## **Features**

### Role-Based Access

- **Normal User**
  - Signup and Login
  - View list of stores
  - Submit or modify ratings (1-5)
  - Logout

- **Store Owner**
  - Login
  - Dashboard showing average ratings for their store
  - View users who submitted ratings
  - Logout

- **System Administrator**
  - Login
  - Add new stores, users, and admins
  - Dashboard showing total users, stores, and ratings
  - View and filter all listings
  - Logout

---

### **Professional UI**

- Soft gradient backgrounds for login, signup, and dashboards  
- Card-style store listings  
- Submit/Modify rating buttons fully functional  
- Role-based redirection and protected routes  
- Responsive design for desktop and tablet
